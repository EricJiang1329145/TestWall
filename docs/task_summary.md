#### 前端实现
- **分页状态管理**：使用 `currentPage`、`pageSize` 和 `totalMessages` 状态变量
- **动态获取页面大小**：通过 `/api/get_page_size` 接口获取后端配置的页面大小
- **分页参数计算**：根据当前页码计算 `start` 和 `end` 参数
