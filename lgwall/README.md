# 校园墙Vue应用

这是一个用于获取并显示校园墙帖子数据的Vue应用。

## 功能

- 通过访问 `https://www.rz101.com/api/get_messages` 获取帖子数据
- 直接显示返回的JSON内容
- 提供刷新数据功能
- 显示加载状态和错误处理

## 技术栈

- Vue 3 (TypeScript)
- Vite
- Tauri

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 构建

```bash
# 构建应用
npm run build
```
